## Что делает скрипт

Windows 10 LTSC поставляется без приложений Xbox / Gaming Services, однако ОС всё равно пытается открывать ссылки `ms-gamingoverlay://`. Это вызывает всплывающее окно с предложением найти в Магазине приложение для обработки протокола. Скрипт подавляет это окно следующим образом:

1. Отключает **GameDVR** и **GameBar** через ключи реестра в `HKCU` и `HKLM`.
2. Удаляет привязку протокола `ms-gamingoverlay` из реестра, чтобы Windows прекратила искать обработчик.
3. Удаляет остаточные временные файлы пакетов Gaming Services и Xbox App.

## Требования

- Windows 10 LTSC (любая сборка) - Автор тестировал на Windows 10 IoT 21H2
- **Права администратора** (скрипт проверяет наличие прав и завершает работу при их отсутствии)

## Использование

1. Нажмите правой кнопкой мыши на `fix.bat` → **Запуск от имени администратора**.
2. Скрипт закроет Проводник, применит изменения реестра, очистит временные файлы и автоматически перезапустит Проводник.
3. Нажмите любую клавишу при появлении запроса, чтобы закрыть окно.

## Что изменяет скрипт

| Раздел реестра | Параметр / Действие | Значение |
|---|---|---|
| `HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\GameDVR` | `AppCaptureEnabled` | `0` |
| `HKCU\System\GameConfigStore` | `GameDVR_Enabled` | `0` |
| `HKCU\System\GameConfigStore` | `GameDVR_FSEBehaviorMode` | `2` |
| `HKCU\System\GameConfigStore` | `GameDVR_FSEBehavior` | `2` |
| `HKLM\SOFTWARE\Policies\Microsoft\Windows\GameDVR` | `AllowGameDVR` | `0` |
| `HKLM\SOFTWARE\Microsoft\PolicyManager\...` | `value` | `0` |
| `HKCU\SOFTWARE\Classes\ms-gamingoverlay` | *(раздел удалён)* | — |

Удаляемые временные файлы: `%localappdata%\Packages\Microsoft.GamingServices_*` и `Microsoft.XboxApp_*`.

## Откат изменений

Изменения реестра можно отменить вручную через `regedit.exe`. Системные файлы не затрагиваются — изменяются только ключи реестра пользовательского уровня и политики, а также временные данные текущего пользователя.
